微信搜一搜35%用户通过搜索发现新小程序，抖音"性格测试"话题播放量超500亿次。**内容不是成本，是ASO排名引擎的燃料。** 每篇推送都是一次搜索权重积累。

### 内容三角模型

```
          人格科普 (周一)
          /  权威信任  \
         /    品牌锚    \
        /               \
   用户故事 (周三) —— 趣味测试 (周五)
   情感共鸣+社会证明    轻量互动+裂变钩子
```

- **人格科普**：建信任。用户因为"AI准"而来，因为"理解原理"而留
- **用户故事**：建共鸣。社会证明>品牌广告，真实故事>营销话术
- **趣味测试**：建裂变。轻量零门槛，最适合社交传播

## 附：内容复用与迭代

每周日复盘4个问题：

1. **哪篇CTR最高？** → 放大该标题/选题风格到下周
2. **哪个关键词搜索排名提升最多？** → 下周内容中强化该关键词覆盖
