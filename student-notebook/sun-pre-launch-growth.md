## 一、双平台搜索环境分析

### 1.1 微信搜一搜 — 小程序搜索生态

| 维度 | 数据/判断 | 对掌心人格局的影响 |
|------|-----------|---------------------|
| 月活用户 | 微信搜一搜 MAU ~9亿 (2025Q4公开数据) | 搜索是最大的免费获客渠道 |
| 小程序搜索占比 | 约35%微信用户通过搜索发现新小程序 | 关键词配置是上线第一优先级 |
| 心理/性格类搜索量 | 2025年"性格测试"微信指数均值 ~800万/日 | 赛道搜索需求旺盛 |
| 关键词限制 | 小程序后台可配10个关键词，36字符 | 每字符都要精打细算 |
| 搜索排名因子 | 名称匹配度 > 描述匹配度 > 使用量 > 留存 | 名称里的关键词权重最高 |
| 长尾词红利 | 微信搜一搜长尾词竞争度低，转化率高 | "手掌拍照测性格"类精准词性价比高 |
| 服务搜索 | 支持服务直达（搜索直接进功能页） | 可配置"拍手掌"直达拍照页 |

**结论**：微信侧的ASO策略应以**名称关键词 + 10个精准关键词 + 长尾词覆盖**三层结构展开，优先争夺"性格测试""人格测验"等中高热词，同时用精准长尾词（"AI手掌分析"等）抢占蓝海。

### 1.2 抖音小程序 — 搜索+内容双驱动

| 维度 | 数据/判断 | 对掌心人格局的影响 |
|------|-----------|---------------------|
| 搜索DAU | 抖音搜索DAU ~6亿 (2025公开数据) | 搜索量级超过微信 |
| 小程序发现方式 | 短视频挂载 > 搜索 > 直播 > 个人主页 | 内容驱动是抖音的核心增长飞轮 |
| 心理测试类内容 | "性格测试"话题播放量超500亿次 | 内容生态成熟，但需找到差异化切入点 |
| 关键词配置 | 抖音小程序标题+描述+标签 | 标签比微信更灵活但权重更分散 |
| 推荐算法 | 兴趣推荐+使用频率+评分 | 冷启动需要种子用户和内容矩阵 |
| 竞争格局 | 抖音心理测试小程序数量远超微信 | 竞争更激烈，但流量盘子更大 |
| 内容挂载 | 视频左下角直接挂小程序入口 | 需要配合达人/自产短视频推广 |

**结论**：抖音侧策略应以**短视频内容矩阵 + 差异化AI手掌标签**为核心。不能只做搜索ASO——抖音的搜索是"搜索+推荐"混合，必须配合内容产出。

## 二、30个ASO关键词三维评分

### 评分体系

每个关键词在三个维度上1-10分评分：

- **热度 (Heat)**：基于行业公开数据的搜索量估算（1=长尾冷门，10=顶级热度）
- **竞争度 (Competition)**：搜索结果数/同类小程序数的反向指标（1=蓝海无竞争，10=红海血拼）
- **相关度 (Relevance)**：与掌心人格局产品核心功能的匹配度（1=弱相关，10=完美匹配）

**综合分 = 热度×0.3 + (10-竞争度)×0.3 + 相关度×0.4** — 相关度权重最高（ASO转化的核心是意图匹配）

### 关键词分级

```
🏆 核心词 (Tier 1): 综合≥7.0 — 小程序名称+描述必须覆盖
🥈 扩展词 (Tier 2): 综合6.0-6.9 — 10个关键词位优先
🥉 长尾词 (Tier 3): 综合5.0-5.9 — 内容/社区覆盖
```
