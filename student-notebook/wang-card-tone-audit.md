# 6张报告卡片语调温度审查

## 审查方法

逐张卡片读取源码（`persona-templates.ts` + `resonance-narrative-engine.ts` + `reportUtils.ts` + `toneVariants.ts`），标注每张卡片的实际温度范围，对比设计规范（关系洞察温3-5° / 彩蛋温4-5° / 洞见温3-5° 温和 or 热6-7° 犀利 / 建议温3-4° / 徽章温3-4° / 广告钩温4-6°），标记偏差。

温度标尺：
```
冷0-2°: 客观数据 / 温3-5°: 朋友语气 / 热6-7°: 尖锐洞察 / 烫9-10°: 冒犯
```

## Card 1: 关系频率密码

**源码位置**: `persona-templates.ts:generateRelationshipCode()` (L335-386)
**设计温度**: 温4-5°

### 现状

| 子字段 | 变体数 | 实际温度 | 判断 |
|--------|--------|---------|------|
| frequencyLabel | 3 | 温4-5° | ✅ 适中 |
| signalPattern | 4 | 温5° | ✅ 适中 |
| bestMatchType | 4 | 温4-5° | ✅ 适中 |
| tensionPoint | 4 | 热6-7° | ⚠️ **偏烫** |

### 具体问题

**tensionPoint "底线太清楚"**
> "这不是缺点——但让在乎的人知道你的底线从哪来，比只亮出底线更有用。"

"只亮出底线"有轻微的指责语气——用户看到第3张卡片时防御已经降低，这里应该更柔和。

**tensionPoint "太能忍"**
> "学会在不满还没变成'算了'之前说出来，是你要做的功课。"

"你要做的功课"是教师语气（热6°），不是朋友语气（温4°）。在首次报告场景，用户还没建立足够信任来接受这种直接建议。

**tensionPoint "太关注自己"**  
> "有时候太关注自己的感受而忘了对方也有他的频率。"

是4条tension中最烫的（热7°）——直接批评用户"太关注自己"。首次报告不应该出现。

### 改进建议

```diff
- '你在关系里最容易出问题的地方是：底线太清楚了。对方可能觉得你"太硬"，但其实你只是不擅长在重要的事情上妥协。这不是缺点——但让在乎的人知道你的底线从哪来，比只亮出底线更有用。'
+ '你在关系里最让人意外的地方是：你看起来很好说话，但有自己绝不会让的线。对方可能一开始没注意到——直到某天触到那条线才发现。这不是问题，是你的保护机制。'

- '你在关系里最容易出问题的地方是：太能忍了。你以为忍过去就好，但其实每一次忍都在身体里存着。学会在不满还没变成"算了"之前说出来，是你要做的功课。'
+ '你在关系里藏得最深的地方是：忍。你以为忍过去就好了，但身体替你记住了每一次。试着在"算了"之前说一句"我在意"——不用大声，对方能听到就行。'

- '你在关系里最容易出问题的地方是：有时候太关注自己的感受而忘了对方也有他的频率。你们不是同步率不够，是频率没对上。最好的关系不是完全一样的人，而是愿意互相调频的人。'
+ '你在关系里最大的挑战是：有时候太在意自己的感受了——不是自私，是你太想知道"我在这段关系里是什么位置"。放轻松，最好的关系不是两个完全一样的人，是愿意为对方调一下频率的人。'
```

## Card 2: 名人彩蛋

**源码位置**: `persona-templates.ts:CELEBRITY_MAP` (L264-325)
**设计温度**: 温4-5°（轻量、趣味）

### 现状

12人格 × 3名人的36条reason文案温度分布：

| 温度 | 数量 | 示例 |
|------|------|------|
| 温3-4° 温暖共鸣 | 18 | "可靠而温暖——他演的角色总能让人放心，像你一样" |

| 热6° 轻度刺痛 | 8 | "复杂不是你的问题，是别人的"、"一辈子在走、在看、在写" |

### 具体问题

名人彩蛋是报告的第5张卡片——用户已经看过核心真相、洞察、关系密码，到了彩蛋部分应该轻松愉快。但有些reason偏尖锐：

- **deep-thinker → Nolan**: "复杂不是你的问题，是别人的" — 热6°，在彩蛋场景太冲
- **sharp-pioneer → Jobs**: "你的'做了再说'和他的'现实可以改变'是同一种东西" — 积极但微热

### 改进建议

```diff
- '他的电影需要看三遍才懂——复杂不是你的问题，是别人的'
+ '他的电影需要看三遍才懂——深度不是缺点，是门槛'

- '对艺术的偏执是你和他的共同语言'
+ '对作品有自己不愿妥协的标准——这一点你们很像'

- '一辈子在走、在看、在写——你的自由灵魂是她那种不被任何地方困住的人'
+ '一辈子在走、在看、在写——她的自由里有一种你也会懂的诗意'
```

### 具体问题

扫描12类型insightPool，发现以下偏烫条目（温和版不应出现热6+）：

| 人格类型 | 偏烫文案 | 问题 |

| silent-guardian | '不要总是把别人的需求放在自己前面' | 否定句式，指导语气 |
| sharp-pioneer | '带领他人之前，先确认方向' | 说教语气 |

| wind-wanderer | '偶尔规划一下，会让自由更有方向' | 指导语气 |

```diff
# bridge-builder

- '有时候说出来，会让别人更懂你'
```

## Card 4: 每周建议

**源码位置**: `persona-templates.ts:assembleReport()` L427
**设计温度**: 温3-4°

### 现状 ⚠️ 严重问题

```typescript
weeklyAdvice: weeklyAdvice ?? '这周试着做一件事：在说"好的"之前，先停3秒问问自己"我想不想"。',
```

**所有12种人格类型共享同一条每周建议**。这是一个结构性缺陷。

温度审视：当前唯一建议温4°，温度本身OK，但完全不区分人格类型意味着：
- 对"锐意开拓者"无用（他们的问题不是不会说"不"，是冲太快）
- 对"静默山峰"不适用（他们的问题是太沉默，不是说"好的"太多）
- 对"根系守护者"部分适用

### 改进建议

为每种人格类型生成不同的每周建议（基于人格特征）：

| 人格类型 | 每周建议（温3-4°） |
|---------|-------------------|
| starry-dreamer | 这周试着把脑子里那个想法写下来——不用完美，写下来就行。 |
| silent-guardian | 这周试着对在乎的人多说一句"我也是"——他们需要听到。 |
| flame-explorer | 这周试着把一件事做到结束，而不只是开始。从最小的那件做起。 |
| deep-thinker | 这周试着做一个"不用分析"的决定——就跟直觉走一次。 |
| gentle-healer | 这周试着先说"我需要"而不是"你需要什么"。顺序很重要。 |
| sharp-pioneer | 这周试着留10分钟什么也不做——不计划、不推进、不等结果。 |
| moon-artist | 这周试着把一个作品发给一个人，不问评价，只发。 |
| bridge-builder | 这周试着不主动调解一次——看看事情自己能不能好。 |
| quiet-mountain | 这周试着多说一句——就一句。你会发现那句话本来就在嘴边。 |
| sunshine-spark | 这周试着让某个人看到你没在笑的样子。你不需要永远明亮。 |
| wind-wanderer | 这周试着在一个地方多待半小时——看看有没有新发现。 |
| root-keeper | 这周试着做一件计划外的事——不需要准备，想到就做。 |

**实现**: 在 `PersonaTemplate` 接口新增 `weeklyAdvice` 字段，或在 `assembleReport` 中根据 `template.type` 查表。

## Card 5: 身份徽章

**源码位置**: `resonance-narrative-engine.ts:buildIdentityBadge()` L129-143
**设计温度**: 温3-4°（中性标签）

### 现状

3个变体，随机选择：
```
1. 「{label}」—— 一个{trait}的人
2. {label} · {trait}型
3. 手掌形塑的 · {label}
```

温度温3°，温度本身OK，但表现力不足。

### 具体问题

- 变体3"手掌形塑的 · {label}"丢了{trait}信息——这是最有辨识度的部分
- 徽章只有格式变化，没有策略变化——应该有时温暖、有时锐利
- 没有利用keywordPool中的3个关键词（"浪漫·直觉·创造力"）

### 改进建议

```diff
const badges = [
-   `「${template.label}」—— 一个${trait}的人`,
+   `「${template.label}」—— ${trait}，只是你的一面`,
-   `${template.label} · ${trait}型`,
+   `${template.label} · ${trait}`,
-   `手掌形塑的 · ${template.label}`,
+   `手掌形塑的 · ${trait}灵魂`,
];
```

**源码位置**: `persona-templates.ts:AD_TEASER_TEXTS` L241-247
