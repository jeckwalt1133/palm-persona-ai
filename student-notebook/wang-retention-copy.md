# 留存文案设计：6条推送消息

## 设计原则

**借鉴来源**: Claude Routines 主动服务模式
- Claude Routines 的核心：在用户需要的时候主动出现，而不是等用户想起来
- 每条推送必须回答：用户凭什么要点开？凭什么要回来？

**方法论**（沿用3模块）:
- M01 情感锚点 — 每句话绑定具体行为/场景
- M02 温度控制 — 留存推送用温4-5°，不冷不热不推销
- M03 消费级语言 — 不说"AI分析完成"，说"手掌说了话"

**推送策略**:
- 频率：7天未活跃触发第1条，10天第2条，14天第3条（阶梯唤醒）
- 时段：晚上8-10点（用户空闲+情绪最开放）
- 微信服务通知（小程序订阅消息）为主，短信备用

---

## 一、7天未使用推送（3条阶梯唤醒）

### 推送1: 「7天 · 轻提醒」

**触发条件**: 用户7天未打开小程序
**推送渠道**: 微信服务通知
**温度**: 温4° / 像朋友随口一提

```
📱 掌心人格局
━━━━━━━━━━━━
你上次拍的手掌，
掌心还有话没说完。

7天前AI从你的掌纹里读到了一些东西——
可能你自己都没注意到。

👉 回来看一眼，1分钟。
```

**M01 锚点**: 「上次拍的手掌」「7天前」— 具体行为+时间锚定
**M02 温度**: 温4° — 不催不逼，像朋友随口问
**M03 翻译**: 不说「重新分析」，说「掌心还有话没说完」

---

**触发条件**: 推送1未唤醒，10天未活跃
**推送渠道**: 微信服务通知
**温度**: 温5° / 朋友之间的小秘密

```
📱 掌心人格局
━━━━━━━━━━━━
你知道每天有多少人
第一次通过手掌认识自己吗？

昨天是300多个。
其中7个和你可能是同一种人格。

👉 你的「同类」在等你。
```

**借鉴来源**: Claude Routines "Your weekly summary is ready" + 豆包社区感
**M01 锚点**: 「昨天是300多个」「7个和你同一种人格」
**M02 温度**: 温5° — 制造归属感，不制造焦虑
**M03 翻译**: 不说「DAU数据」，说「300多个人」「你的同类」

### 推送3: 「14天 · 新鲜钩子」

**触发条件**: 推送2未唤醒，14天未活跃

**M03 翻译**: 不说「功能更新」，说「这次可能会对你说真话」

## 二、分享回流推送（2条）

### 回流1: 「朋友扫码后」即时推送

**触发条件**: 分享者的报告被朋友扫码查看
**推送渠道**: 微信服务通知
**温度**: 温5° / 温暖社交信号

```
📱 掌心人格局
━━━━━━━━━━━━
有人刚看了你分享的报告。

你那份「星辰梦想家」的身份卡，
被另一个人认真读了一遍。

ta 可能正在犹豫要不要也拍一张。
👉 问问 ta 看到了什么？
```

**借鉴来源**: Claude Teleport概念 — A的分享→B的体验是连续故事
**M01 锚点**: 「星辰梦想家」「刚看了」「认真读了一遍」
**M02 温度**: 温5° — 社交温暖，不是push推销
**M03 翻译**: 不说「转化漏斗」，说「可能在犹豫要不要也拍一张」

**设计逻辑**: 
- 不说「邀请ta来测」→ 让分享者自然聊天
- 给分享者一个社交货币：「你的身份卡被认真读了」
- 为后续自然对话搭桥：「问问ta看到了什么」

---

### 回流2: 「多个朋友扫码后」聚合推送

**触发条件**: 分享者的报告被≥3个不同朋友扫码
**推送渠道**: 微信服务通知
**温度**: 温5° / 你的眼光被验证了

```
📱 掌心人格局
━━━━━━━━━━━━
已经有3个人
通过你的分享走进了掌心人格局。

其中2个人测出了
和你完全不一样的人格。

这说明你身边的人都挺特别——
或者说，你挑朋友的眼光很准。
👉 看看谁测了？你们像不像？
```

**借鉴来源**: Claude Code Analytics API — 量化展示影响力
**M01 锚点**: 「3个人」「2个人测出了不一样的」
**M02 温度**: 温5° — 「你挑朋友的眼光很准」是正向强化
**M03 翻译**: 不说「分享转化率」，说「你挑朋友的眼光很准」

**设计逻辑**:
- 把数据变成社交货币——「3个人因为你来了」
- 制造好奇心——「完全不一样的人格」让人想看
- 不催不推——让用户自己好奇

## 三、人格变化提醒（1条）

### 变化1: 「再次测试结果不同」

**触发条件**: 用户≥7天后再测，人格类型与上次不同
**推送渠道**: 微信服务通知
**温度**: 温4° / 好奇中带一点认真

```
📱 掌心人格局
━━━━━━━━━━━━
AI 这次读出了不一样的你。

上次是「星辰梦想家」，
这次是「火焰探索者」。

手掌没变，
是这两个星期里你变了一些东西。
或者说——你展现了另一面。

👉 看看到底哪里不一样？
```

**借鉴来源**: Claude Memory for Agents — 跨会话状态追踪
**M01 锚点**: 「上次是星辰梦想家」「这次是火焰探索者」「这两个星期」
**M02 温度**: 温4° — 「你变了一些东西」或「展现了另一面」，不给判断
**M03 翻译**: 不说「人格类型变更」，说「AI读出了不一样的你」

**设计逻辑**:
- 最关键的一条——可能引发焦虑（"我变了？是不是不准？"）
- 用「展现了另一面」而不是「你变了」——降低防御
- 「手掌没变」→ 强化物理锚点，增加可信度
- 两个选项都给：「你变了」或「你展现了另一面」——让用户自己选
