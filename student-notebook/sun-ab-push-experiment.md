# 推送文案A/B实验方案 — 情感共鸣 vs 数据价值

## 1. 实验背景与行业基准

### 1.1 为什么推送文案是当前最高ROI的增长杠杆

竞品扫描显示行业已进入**第三阶段——价值证明期**：
- 豆包/Kimi/文心三家同步启动付费（68元/月价格锚点）
- Grok付费失败教训：没有不可替代的AI价值，用户不会买单
- OpenAI+Anthropic联手切企业市场——C端增长从"烧钱获客"转向"留存变现"

对掌心人格局而言，**推送是唯一主动触达用户的合法渠道**（小程序无朋友圈分享外链、无PUSH权限、无App安装包）。一条推送文案的质量，直接决定当日DAU的20-40%。

### 1.2 行业推送基准数据

| 平台 | 行业平均推送CTR | 头部产品CTR | 推送打开后当日留存 |
|------|----------------|-----------|-----------------|
| 微信小程序订阅消息 | 3-8% | 12-18% | 35-50% |
| 抖音小程序模板消息 | 2-5% | 8-12% | 25-40% |
| AI工具类APP Push | 4-10% | 15-25% | 30-45% |

来源：微信公开课2025数据、抖音开放平台运营指南、36氪AI应用增长报告

掌心人格局当前无推送系统（V7规划中），本次实验同时验证**推送基建+文案策略**两个假设。

### 1.3 竞品推送风格扫描

| 竞品 | 推送风格 | 典型文案 | 推测CTR |
|------|---------|---------|---------|
| 豆包 | 价值驱动 | "你的AI助手又学了新技能，来看看" | 8-12% |
| Kimi | 好奇心钩子 | "你上次问的那个问题，我有了新答案" | 10-15% |
| DeepSeek | 功能提醒 | "V4模型已上线，推理能力提升40%" | 5-8% |
| 测测(竞品) | 情感共鸣 | "你的性格画像更新了——有些变化你自己都没发现" | 12-18% |

**核心发现**：人格/心理类产品的推送CTR显著高于工具类。情感共鸣型推送在竞品中表现更优。

## 2. 实验假设

| # | 假设 | 理论依据 |
|---|------|---------|
| **H1** | 情感共鸣型推送的CTR高于数据价值型（预期差异≥30%） | 人格类产品用户动机是"了解自己"，情感钩子比数据钩子更匹配 |
| **H2** | 数据价值型推送带回的用户，会话深度更高（停留时长+报告完成率） | 被"数据/维度"吸引回来的用户有更强的分析需求 |
| **H3** | 情感共鸣型的7日回访率高于数据价值型 | 情感共鸣建立的心理连接比数据驱动更持久 |

**零假设**：两种风格在所有指标上无显著差异。

## 3. 实验设计

### 3.1 通用参数

```
实验类型:   A/B 随机对照
平台:       微信小程序订阅消息 + 抖音小程序模板消息（双端同步）
随机单位:   用户级（同一用户始终收到同一风格）
流量分配:   50/50
最小检测效应: Cohen's d ≥ 0.3
显著性水平:  α = 0.05
统计功效:   1-β = 0.80
实验周期:   14天（覆盖2个完整周，排除周末效应）
```

### 3.2 自变量：推送文案风格

**风格A — 情感共鸣型** (Emotional Resonance)

核心理念：用"被理解的感觉"驱动打开。不强调功能，强调感受。

文案配方：
- L1锚点（具体情境） + 情感钩子（"你也是吗"） + 好奇心缺口（"AI读出了什么"）
- 温度区间：温5°～热7°（王富贵温度理论）
- 关键词：发现、秘密、懂你、不一样、原来

**风格B — 数据价值型** (Value-Driven)

核心理念：用"明确的收益预期"驱动打开。强调功能和数据维度。

文案配方：
- 维度数量 + 具体特征 + 可操作价值（"看看你的"）
- 温度区间：温3°～温5°（理性但不过冷）
- 关键词：数据、维度、报告、分析、特征、指数

### 3.3 因变量（度量指标）

| 层级 | 指标 | 定义 | 测量方式 |
|------|------|------|---------|
| **P0 主指标** | **DAU Lift** | 实验期内日均DAU / 实验前7天基线DAU - 1 | 服务端DAU日志 |
| **P0 主指标** | **推送CTR** | 点击推送 / 推送送达 | 微信/抖音推送回调 |
| **P1 次指标** | **会话深度** | 推送带回用户的平均页面浏览数 | 前端埋点session_id |
| **P1 次指标** | **报告完成率** | 推送带回用户中完成完整报告的占比 | 服务端报告生成日志 |
| **P1 次指标** | **7日回访率** | 推送触达后7天内再次自然打开的比例 | 服务端DAU日志匹配 |
| **P2 观察指标** | **分享率** | 推送带回用户中触发分享的比例 | 前端分享埋点 |
| **P2 观察指标** | **退订率** | 推送后关闭订阅的用户比例 | 订阅状态变更日志 |

### 3.4 控制变量

| 变量 | 控制方式 |
|------|---------|
| 推送时间 | 两组同时发送（每日18:00 UTC+8） |
| 推送频次 | 相同（每48h一条，实验期共7条） |
| 推送模板 | 相同（微信订阅消息同一模板ID） |
| 落地页 | 相同（报告首页，无差异化） |
| 用户群 | 随机分配，事前检验两组在基线DAU/活跃度上无显著差异 |
| 文案长度 | 控制在25-40字（微信订阅消息限制） |

### 3.5 样本量估算

```
目标: 检测两组CTR差异（预期A=12%, B=8%, 差异4pp）
效应量: h=0.25（中小效应，Cohen's h）
α=0.05, 1-β=0.80

每组所需样本: n ≈ 250（卡方检验两样本比例）
考虑30%推送未送达率: n ≈ 360/组
考虑20%实验期内流失: n ≈ 450/组

总计: 900订阅用户
当前订阅用户基数预估: ~200（需先完成推送基建+订阅引导）
缺口: 需在实验前完成一轮订阅引导增长（目标+250订阅）

实验前备选方案（若订阅数<600）:
  → 按序贯检验设计（sequential testing），每50个样本做一次中期分析
  → 允许在达到显著性后提前终止
```

## 4. 测试文案（7组对照）

每组对应一次推送。两版文案基于同一推送主题，仅风格不同。

### 第1组：报告生成通知

**A-情感共鸣**:
> 你的手掌图像分析完成了。AI说——你身上有3个特质，你自己可能都没注意到。看看准不准？

**B-数据价值**:
> 专属人格报告已生成：5大维度 × 12项特征指数完整分析。查看你的掌心人格画像 →

### 第2组：好友匹配提醒

**A-情感共鸣**:
> 有人想和你测测默契度。说真的，有时候AI比你自己更早发现你们哪里合拍。

**B-数据价值**:
> 1份新匹配邀请：基于双方手掌特征的同频度+互补度分析。查看匹配报告 →

### 第3组：沉睡唤醒

**A-情感共鸣**:
> 3天没见你了。你的手掌里还藏着一个没看完的故事——上次分析到一半的那个。

**B-数据价值**:
> 你的掌心人格数据有更新：上次未完成的报告已保存。继续分析 →

### 第4组：新洞察推送

**A-情感共鸣**:
> 重新看了一遍你的手掌——发现一个有意思的细节。你表达情绪的方式，比你以为的更有层次。

**B-数据价值**:
> 人格洞察更新：情绪表达维度新增2项特征识别，准确度提升至92%。查看更新 →

### 第5组：周末深度推荐

**A-情感共鸣**:
> 周末了，花3分钟了解一个你可能一直在回避的问题：你最擅长的那种"坚强"，其实叫什么。

**B-数据价值**:
> 周末深度报告推荐：防御机制维度完整解读 + 3条基于你手掌特征的改善建议。开始阅读 →

### 第6组：社交裂变钩子

**A-情感共鸣**:
> 你的3个朋友可能和你是同一种人格类型。不是巧合——是某种你们自己都没发现的共同频率。

**B-数据价值**:
> 掌心人格匹配功能开放：基于6项核心特征指数，发现与你同频度≥85%的好友。立即匹配 →

### 第7组：周报总结

**A-情感共鸣**:
> 这一周，你查看了3次自己的人格报告。你在找什么答案？——也许AI已经读出来了。

**B-数据价值**:
> 本周回顾：3次分析 × 5个维度变化 × 2项新洞察。查看你的掌心人格周报 →

## 5. 成功标准

### 5.1 主要判定逻辑

| 场景 | 条件 | 结论 | 行动 |
|------|------|------|------|
| **A显著胜出** | A在CTR上显著高于B（p<0.05）且DAU Lift≥+10% | H1成立，情感共鸣型为默认推送风格 | 全量切换A风格；保留B作为功能型推送备用 |
| **B显著胜出** | B在CTR上显著高于A（p<0.05）且DAU Lift≥+10% | H1不成立，数据价值型更有效 | 全量切换B风格；重新审视情感文案设计方向 |
| **A胜CTR但B胜深度** | A的CTR更高但B的会话深度+报告完成率更高 | H1+H2部分成立，需分层策略 | 日常推送用A，功能/报告通知用B |
| **无显著差异** | 所有指标p>0.05 | H0成立，文案风格对DAU无显著影响 | 扩大样本复现；或测试其他变量（推送时间/频次/图片vs纯文） |

### 5.2 定量阈值

| 指标 | 基线（预期） | A组目标 | B组目标 | 判定"差异显著"阈值 |
|------|-----------|---------|---------|-----------------|
| 推送CTR | 5-8% | ≥12% | ≥8% | 组间差异≥3pp 且 p<0.05 |
| DAU Lift | — | ≥+15% | ≥+10% | 组间差异≥5pp |
| 会话深度(页) | 2.5 | ≥3.5 | ≥4.0 | 组间差异≥0.5页 且 p<0.05 |
| 7日回访率 | 20% | ≥30% | ≥25% | 组间差异≥8pp |
| 退订率 | <2% | <3% | <3% | 任一组>5%触发安全审查 |

### 5.3 安全终止线

```
- 任一组退订率 > 5% → 暂停该组推送，审查文案是否过度
- 任一组用户投诉 ≥ 3起 → 全实验暂停，合规审查
- 实验第7天进行中期分析，若一组CTR < 3% → 该组文案重设计
```

## 6. 埋点需求

```
事件: push_send           — 推送发送（含style: A/B, push_id, user_id）
事件: push_delivered      — 推送送达（微信/抖音回调确认）
事件: push_click          — 推送点击（含style, push_id, user_id）
事件: push_session_start  — 推送带回的会话开始（含source: push, style, push_id）
事件: push_session_depth  — 推送会话的页面浏览数（会话结束时上报）
事件: push_report_complete— 推送带回用户完成了报告生成
事件: push_share          — 推送带回用户在本次会话中触发分享
事件: push_unsubscribe    — 用户在推送后24h内关闭订阅
```

微信订阅消息特殊处理：
- `push_delivered` 在微信侧不可直接获取（微信不返回送达状态），以 `push_send` 为分母计算CTR
- 抖音模板消息可通过回调获取 `push_delivered`

## 9. 风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| 订阅用户数不足 | **高** | 实验无法启动或统计功效不足 | 序贯检验备选方案；实验前订阅引导增长冲刺 |
| 微信订阅消息模板审核不通过 | 中 | 延迟实验启动 | 提前2周提交模板审核；准备备用模板 |
| 推送被微信限频/折叠 | 中 | CTR全线偏低 | 控制48h间隔；避免触发微信垃圾推送检测 |
| 实验期间有竞品重大发布影响用户行为 | 低 | 数据被外部事件污染 | 实验日志记录外部事件时间线；分析时标注+敏感性检验 |
| 两组基线不均衡 | 低 | 归因偏差 | 事前AA检验：随机分组后在实验Day 1仅记录不推送，验证两组自然DAU无显著差异 |
| 情感共鸣型文案踩合规红线 | 低 | 产品下架风险 | 所有文案经合规门禁(ComplianceGate)预检；禁用词扫描 |

## 11. 附录：与王富贵温度理论的对齐

本实验的两种推送风格可映射到王富贵的温度框架：

| 维度 | 情感共鸣型 (A) | 数据价值型 (B) |
|------|---------------|---------------|
| 温度区间 | 温5°～热7° | 温3°～温5° |
| L1锚点 | 具体情境/感受描述 | 具体数字/维度/百分比 |
| L2共鸣 | "你也是吗"式邀请共鸣 | "你的数据在这里"式事实陈述 |
| L3释然 | "原来如此"式自我理解 | "数据告诉你"式客观解读 |
| 适配场景 | 日常唤醒、情感连接、裂变 | 报告通知、功能提醒、数据更新 |
| 预期风险 | 过热→退订（需监控冒犯率） | 过冷→忽略（需监控CTR底线） |

本实验完成后，可与王富贵的实验1（冷2°vs温5°）结果交叉分析，验证温度理论在推送场景是否同样成立。
