## 优先级决策逻辑

### 为什么P0是分享+模型+合规（而不是Agent产品化）？

1. **分享闭环** = 增长引擎。没有增长，产品做再好也没人用。对比豆包的免费圈地→付费转化，我们更需要先有用户。
2. **模型升级** = 成本+体验双赢。DeepSeek V4的$0.30/MTok是V3的1/50，且开源Apache 2.0意味着无法律风险。
3. **合规上线** = 前置条件。微信审核不通过，一切归零。
4. Agent产品化虽然趋势明确（Claude Managed Agents），但需要分享闭环跑通后才有用户来体验Agent透明度。

### 为什么记忆Phase 2是P1而不是P0？

三个信号（Claude Memory / GPT-5.5记忆 / DeepSeek Engram）都说"记忆是标配"，但：
- 首次用户体验（拍照→报告→分享）比"回来看看变化"更优先
- 记忆需要至少2次测试才有价值——先让用户完成第1次
- Phase 1基础设施已就位，Phase 2可快速启动
