# 模块1：文案共鸣基础 — 让用户愿意分享的文字

## 学习目标

完成本模块后，你能够：
1. 判断一段文案是否"有共鸣"（不是凭感觉，有判断标准）
2. 把一个"正确的文案"改写成"有共鸣的文案"
3. 为5种基本人格类型各写一句分享文案

## 前置知识

- 完成了 onboarding Day 1
- 理解项目的5种人格维度

## 核心概念

### 什么是有共鸣的文案？

```
正确但不共鸣：  "这是一份基于AI的人格分析报告"
有共鸣：        "天哪，这说的不就是我吗？"

正确但不共鸣：  "分享海报可保存到相册"
有共鸣：        "原来我在别人眼里是这样的……"
```

**共鸣的三个层次**：
1. **认知共鸣** — "它说的对"（准确描述了我的特征）
2. **情感共鸣** — "它理解我"（说出了我没说出口的感受）
3. **社交共鸣** — "我想让ta看"（帮我对别人表达了自己）

### 5种人格 × 分享动机

| 人格倾向 | 分享动机 | 文案方向 |
|---------|---------|---------|
| 开放型 | 新奇感 | "有趣/从没想过/原来如此" |
| 进取型 | 成就感 | "优秀/厉害/潜力无穷" |
| 温暖型 | 连接感 | "懂我/暖心/被看见" |
| 独立型 | 独特感 | "与众不同/独一无二/我就是我" |
| 守护型 | 确认感 | "安心/靠谱/果然如此" |

## 练习

### 练习1：判断共鸣层次（10分钟）

读以下3条文案，判断各属于哪个共鸣层次。每题写出你的判断+理由。

```
文案A: "根据算法分析，你的人格类型为开放型，占比67%"
文案B: "你总是第一个说'试试看'的人，然后发现自己打开了一扇新世界的大门"
文案C: "身边有人不懂你？把这张卡片发给他——让数据替你说话"
```

### 练习2：改写练习（20分钟）

把以下"正确的文案"改写成"有共鸣的文案"。至少改写出2个版本，分别针对不同的人格倾向。

```
原始文案（正确但不共鸣）:
"你的性格测试结果：外向85分，开放72分，尽责60分，宜人55分，情绪稳定45分"
```

改写要求：
- 版本1：针对开放型人格（新奇感）
- 版本2：针对温暖型人格（连接感）

把结果记入你的笔记本。

### 练习3：生成你的第一条分享文案（30分钟）

假设你要为你自己的学习成果写一条分享文案（发到朋友圈/微信群）。

步骤：
1. 明确目标读者（谁会看到这条分享？）
2. 选择情感锚点（你想让读者感受到什么？）
3. 写3个草稿版本
4. 自己选出最好的一条
5. 说明为什么选这一条

## 验收标准

- [ ] 练习1：3条判断全部正确，理由自洽
- [ ] 练习2：至少2个版本，每个版本有明确的目标人格
- [ ] 练习3：3个草稿+1个精选+选择理由
- [ ] 所有练习内容写入 `student-notebook/` 下的当日笔记

## 常见问题

**Q: 我写的文案我自己觉得好，但别人觉得不好怎么办？**
这是正常的。共鸣不是"写得好"，是"读者觉得写出了自己的感受"。你需要让至少1个真实的人（或另一个AI实例）读你的文案并反馈。

**Q: 有没有模板可以直接用？**
有。`persona-templates.ts` 中有引言库 QUOTE_TEMPLATES 和视觉锚点文案逻辑——但那是在你理解了共鸣原理之后才应该使用的东西。先理解再套模板。
