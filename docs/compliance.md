# 合规文档（终版）

## 内容合规
- 所有 AI 生成文本先经过 ContentSafety 过滤
- ContentSafety.filterNarrative() 专门过滤 LLM 文案字段
- 所有报告页固定加粗显示免责声明
- 所有绝对化语气强制弱化："注定/必然/一定" → "倾向于/更容易/大概率"
- 所有特征引用须经合规审查：允许"从手掌特征看，你倾向于..."，禁止"你的掌纹证明..."

## 禁止词与替代表达
### 显性禁词
掌纹、手相、看手相、算命、占卜、命运注定、改运、开运、正缘、姻缘测算、旺夫旺妻、克、寿命预测、疾病预测、灾祸预测、财富暴富预测、100%准确、比算命更准、必然、一定会、天生一对、宿命
### 替代表达
手掌图像特征、手部轮廓线条、掌心线条、性格倾向、相处模式、互动适配度、同频度、互补度、趣味语境下、倾向于、更容易、大概率
### 句式改写规则
- "你的掌纹证明" → "从手掌特征看，你倾向于"
- "你命中注定" → "你更容易形成某种互动模式"
- "你一定会" → "你更可能"
- "这是你的命" → "这是根据当前特征的分析"

## 过审必备
- 小程序类目："工具→图片处理"（备选"教育→素质教育"），严禁"生活服务→婚庆"
- 首页：青少年模式 + 隐私协议双弹窗（强制启动时弹出）
- ICP 备案：涉及用户上传图片(UGC)，上线前必须完成
- 开发 appid：微信 touristappid，抖音 testAppId

## 隐私合规
- 首次进入显示隐私同意弹窗
- 用户辅助标点数据（坐标点）仅用于本次分析，不持久化
- 手掌原始图片提取特征后即时删除，仅存加密特征哈希
- 隐私政策明确说明"不存储原始手掌图片"
- 用户可随时删除历史报告及关联数据
- 不收集：IMEI、MAC、通讯录、位置（除非用户主动授权）

## 次数与裂变合规
- 每日免费 3 次分析 + 激励视频解锁（官方允许）
- 禁止强制分享、分享到群解锁、不分享不能继续
- 匹配邀请 7 天有效期，过期提示重新发起
- 匹配只说同频度/互补度/相处适配度，绝不说姻缘/天命/正缘

## 安全
- API Key 仅存环境变量，不提交仓库
- 日志不打印完整 base64、Key、敏感信息
- 所有用户输入做 XSS/注入过滤
- 接口限流：同设备 ID 每分钟最多 3 次 /api/analyze
- Fastify bodyLimit: 10MB
- 并发控制：同 userId 分析请求未来需加锁（TODO 标注）

## 审核注意
- 不出现：算命/占卜/风水/命理/星座运势/姻缘/转运转运 等敏感词
- 类目绝对不能选"生活服务 > 婚庆""生活服务 > 宗教"
- 所有页面固定位置显示免责声明（加粗）
- 青少年模式有明确入口和限制说明
