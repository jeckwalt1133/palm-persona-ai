## 测试流程

### 流程一：首页浏览
1. 进入小程序 → 看到首页"掌心人格局"标题 + "开始分析"按钮
2. 首页底部显示免责声明（加粗，固定位置）
3. 页面底部有"开启青少年模式"入口

### 流程二：AI 分析
1. 点击"开始分析" → 进入拍照/选图页面
2. 选择一张手掌照片 → 预览显示选中图片
3. 点击"开始分析" → 等待分析完成
4. 自动跳转至报告页

### 流程三：报告查看
1. 报告页顶部显示人格类型标签 + 分析日期
2. 五维雷达图（情绪频率/沟通同步/行动互补/信任潜力/摩擦风险）
3. 维度解析卡片（分数 + 标签 + 描述）
4. 综合分析文案
5. 关系洞察条目
6. 今日金句
7. 关键词标签
8. 底部悬念文案（次日留存钩子）

### 流程四：历史报告
- 返回首页再次进入 → 自动显示最新报告

### 流程五：青少年模式
1. 首页开启青少年模式
2. 每日仅可分析 1 次
