# ICP 备案指南

## 备案前提

根据《互联网信息服务管理办法》，涉及用户上传图片（UGC）的线上服务需要完成 ICP 备案。

## 备案步骤

### 1. 服务器与域名准备

- 购买国内云服务器（阿里云/腾讯云/华为云）
- 域名完成实名认证（需与备案主体一致）
- 域名已解析至国内服务器 IP

### 2. 备案主体

| 项目 | 内容 |
|------|------|
| 主体性质 | 个人 / 企业 |
| 所需证件 | 个人：身份证 / 企业：营业执照 |
| 备案地址 | [工信部 ICP/IP 备案系统](https://beian.miit.gov.cn) |

### 3. 备案信息填写

- **网站名称**: 掌心人格局
- **网站域名**: [待定]
- **网站首页标签**: AI 人格分析 | 手掌特征分析工具
- **服务内容**: 工具类图片处理
- **前置审批**: 不涉及（非医疗/教育/新闻/出版等需前置审批类目）

### 4. 备案后流程

1. 将备案号添加至小程序底部
2. 链接至工信部备案查询页面
3. 小程序管理后台提交备案号审核

### 5. 注意事项

- 备案审核时间：个人约 5-20 个工作日
- 备案期间服务可正常使用
- 备案通过后需在网站底部悬挂备案号
- 小程序上线前必须完成 ICP 备案
