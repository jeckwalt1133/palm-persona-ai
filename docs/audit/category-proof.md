# 类目证明材料

## 申请类目：工具 → 图片处理

### 产品定位声明

"掌心人格局"是一款**基于手掌图像的 AI 特征分析工具**，核心功能为：

1. 用户拍摄/上传手掌照片
2. AI 提取手部轮廓与线条特征
3. 生成人格分析报告（情绪频率、沟通同步、行动互补等维度）
4. 可视化展示分析结果（雷达图）

### 类目匹配依据

| 类目要求 | 本产品符合情况 |
|---------|--------------|
| 工具类目下的图片处理工具 | ✓ 核心功能基于用户上传的图片进行处理 |
| 提供明确的图片处理结果 | ✓ 输出人格分析报告与可视化图表 |
| 不涉及医疗诊断 | ✓ 明确标注"仅限娱乐参考，不构成医疗/法律/婚恋建议" |
| 不涉及封建迷信 | ✓ 不使用"掌纹/手相/算命/占卜/命运"等违规表述 |

### 内容合规措施

- 所有分析报告内置免责声明
- 不收集用户原始图片，提取特征后即时删除
- 不使用"命中注定""一定会"等绝对化表述
- 所有分析结果标注为"倾向性分析，仅供娱乐参考"

### 备选类目

如"工具 → 图片处理"未能通过，申请调整为"教育 → 素质教育"：

- AI 分析帮助用户自我认知与性格探索
- 分析维度（情绪频率、沟通同步、行动互补）属于心理学/人格分析范畴
- 结果为性格倾向性描述，非诊断性结论
