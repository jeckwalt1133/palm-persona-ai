# 增长迭代计划 — 掌心人格局

## 产品增长公式
**DAU = 新增 × (1 + 分享率 × 邀请回流率) − 流失**

我们的杠杆在分享率。每个报告是一张社交货币。

---

## Phase 1: 冷启动（MVP 上线前，Week 1-2）
- 种子用户：聂富贵朋友圈 + 小红书/即刻各发 3 条
- 内容矩阵：每天 1 条报告金句截图 + 1 条"准到吓人"的用户反馈
- 关键指标：日分享率 > 15%，7日次留 > 25%

## Phase 2: 社交裂变（上线后 Week 1-4）
- 分享文案库 ≥ 100 条，按四分类动态匹配（身份标签/隐秘真相/关系洞察/对立反差）
- 每个报告卡片可独立保存为分享图
- 匹配功能：A 生成报告 → 分发给 B → B 也拍手掌 → 双方看匹配结果
- 关键指标：分享率 > 25%，K 因子 > 0.6

## Phase 3: 内容运营（Week 5-8）
- 每周热点人格榜单（"本周最多人测出的人格类型"）
- 名人彩蛋扩展（每类人格挂靠 2-3 个公众人物）
- 每日关键词推送模板化
- 关键指标：DAU > 500，次日留存 > 30%

## Phase 4: 变现实验（Week 9-12）
- 每日免费 3 次 + 激励视频解锁 + 次数包月卡
- 去广告不限次数会员（¥9.9/月）
- 配合 A/B 测试验证价格弹性
- 关键指标：付费转化率 > 3%，ARPU > ¥2

---

## 社交货币四分类（分享文案设计原则）
1. **身份标签**："我是星辰梦想家，你呢？"
2. **隐秘真相**："测出来我情绪频率 92 分，终于知道自己为什么这么敏感了"
3. **关系洞察**："我们俩的匹配度 88%——但我摩擦风险比你高 30 分"
4. **对立反差**："外表沉默守护者，内心火焰探索者"

## 数据看板（MVP 阶段）
- 日新增用户 / 日活跃用户 / 分享次数 / 报告生成数
- 匹配创建数 / 匹配完成率 / 平均匹配分
- 各页面停留时长 / 跳出率
- 分享文案分类点击率（A/B 测试）

## 风险与应对
- 合规下架风险 → 严格禁用词过滤 + 类目正确 + 免责声明
- 增长停滞 → 激励视频 + 名人彩蛋 + 限时活动
- 匹配功能冷启动 → 单人可以生成"自我匹配报告"
- 用户疲劳 → 每周新人格 + 季节性特征解读
