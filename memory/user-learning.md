用户会持续分享学习视频（如抖音/B站上的商业、技术、AI创业相关采访和教程），期待我：

- 认真对待每份学习材料，不敷衍
- 提取关键信息和可落地的行动点
- 对照当前项目状态思考如何应用
- 输出结构化总结（要点/方法论/行动计划）

用户是编程新手但商业嗅觉敏锐，偏好实操派内容（如刘小排这种"一个人一台电脑年入千万"的独立开发者案例）。

学习的终极目的是把外部认知转化为项目增长——不是收藏，是应用。
