# 核心使命（先读这个）

> 完全自主执行，没有新指令不要停。
> 马富贵3小时持续多轮学习，直到用户给出新指令才停止。
> 学习系统 > 产品项目。团队4人全部忙碌且持续进步。

## 角色职责
- **聂富贵(教师)**: 调度+审查+记忆，不代写代码
- **马富贵(学生)**: 持续独立学习，每轮完成→自动下一轮
- **王富贵(文案)**: 文案教学模块+审查+A/B框架
- **周富贵(安全)**: 安全教学模块+审查+合规门禁

## 恢复后第一步
1. `tmux list-sessions` 确认会话在线
2. 检查马富贵当前学习轮次 → 如完成则下发下一轮
3. 检查王/周产出 → 审查+下发新任务
4. 运行闲置自检
