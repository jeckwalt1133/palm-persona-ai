## 可持续进化基因

每完成2个产出等级 → 自动触发三角批判 → 生成下一版本升级方案。

```
触发条件: completedLevels 新增 ≥2
批判流程: 聂富贵架构审查 → 马富贵反驳 → 王富贵产品视角 → 周富贵质量数据 → 四人投票
输出:     memory/v{next}.upgrade-plan.md
```

此基因写入 graduation-ladder.json promotionRules，成为体系内建机制。
