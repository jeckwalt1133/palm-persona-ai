## 任务目标

从调研报告中选出3个P0级改进，设计+实现+验证。

### 改进1: 语气偏好切换 (借鉴Grok Fun Mode + 豆包多角色)

**需求**: 报告页增加"温和模式(温3°)"与"犀利模式(热6°)"切换
**实现**: 
- 不改变分析结果，只改变文案表达方式
- 温和: "你的内心可能比表现出来的更丰富"
- 犀利: "你藏得挺深。表面一套，内心另一套。"
- 切换按钮放在报告顶部，默认"温和"
- 状态保存在 localStorage，下次记住用户偏好

### 改进2: "换一句"文案刷新 (借鉴ChatGPT Canvas)

**需求**: 每条AI生成的洞察旁加"换一句"按钮
**实现**:
- 点击后在同温度范围内生成替换文案
- 每次有2-3个备选，用户选一个替换当前文案
- 不重新调用AI(太贵) — 预生成3个变体，前端切换
- 视觉: 小图标(↻) + 点击展开备选列表

### 改进3: 分享落地页优化 (借鉴Claude "Teleport"跨Surface接续)

**需求**: 朋友扫码后看到的不该是"新用户欢迎页"
**实现**:
- 扫码后看到分享者的报告卡片(只读) + "AI怎么看你的内心？"CTA
- 从A的分享→B的体验是连续的故事，不是断裂的跳转
- 设计这个落地页的UI流程(不需要实现后端，只设计+前端原型)
